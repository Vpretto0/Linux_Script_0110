## To add a file:   
``git add (filename)``

## To update a file:   
``git commit -m "added files via upload"`` -> Push edits to staging area   
``git push origin main`` -> Push edits to website   
``git pull origin main`` -> Pull edits from website   
``git status`` -> See what is being ignored
